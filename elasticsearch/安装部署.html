安装部署

elasticsearch

从官网下载安装包，本地解压。然后修改配置文件。

启动elasticsearch

bin/elasticsearch

elasticsearch-head

安装node

brew install node

下载安装elasticsearch-head

git clone git://github.com/mobz/elasticsearch-head.git
cd elasticsearch-head
npm install

修改服务器监听地址

修改elasticsearch-head下Gruntfile.js文件，添加：

connect: {
	server: {
		options: {
			hostname: '0.0.0.0',
			port: 9100,
			base: '.',
			keepalive: true
		}
	}
}

启动命令

grunt server

cerebro

从官网下载安装包，本地解压。

启动命令

bin/cerebro -Dhttp.port=1234 -Dhttp.address=127.0.0.1
