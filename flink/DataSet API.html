DataSet API

DataSet Transformations

Data transformations 将一个或多个DataSets 转换为新的DataSet。程序可以将多个transformations 组合为一个复杂的集合。

Map

Map transformation 在DataSet 的每一个元素上应用用户自定义的map function。

它实现了一个一对一的mapping，每次调用函数必会返回一个元素。

FlatMap

FlatMap transformation 在DataSet 的每一个元素上应用用户自定义的flat-map function。它和map function 的区别是可能为每一个input 元素返回0或任意多个元素。

Iterations

迭代算法出现在许多数据分析的领域，如机器学习或者图分析。这些算法对于实现从大数据中提取有意义的结果至关重要。随着在非常大的DataSet 上运行迭代算法的需求越来越多，越来越需要以大规模并行的方式执行迭代。

Flink 通过定义step function 并将它绑定在一个特别的iteration operator 上来实现迭代算法。iteration operator 有两种：Iterate 和Delta Iterate。它们都在当前迭代阶段重复调用step function，直到达到某个终止条件。
