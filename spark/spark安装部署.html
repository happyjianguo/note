spark安装部署

scala安装

安装命令：brew install scala

安装位置：/usr/local/bin/scala

spark安装

下载spark

http://spark.apache.org/downloads.html

下载后解压，安装位置：/Users/xerxes/spark/

配置环境变量
export SPARK_HOME=/Users/xerxes/spark
export PATH=$PATH:$SPARK_HOME/bin
修改配置文件
cd /Users/xerxes/spark/conf
cp spark-env.sh.template spark-env.sh
vim spark-env.sh

添加：

export SCALA_HOME=/usr/local/Cellar/scala/2.13.1
export SPARK_MASTER_IP=localhost
export SPARK_WORKER_MEMORY=4G

安装livy

下载livy

https://www.apache.org/dyn/closer.lua/incubator/livy/0.6.0-incubating/apache-livy-0.6.0-incubating-bin.zip

下载后解压，安装位置：/Users/xerxes/spark/livy

修改配置文件
  1. livy.conf
  2. livy-env.sh
启动livy
bin/livy-server start
